# Projet 3 : Analyse d'un fichier CSV

**Réalisé par :** Guy Beyina

## Fonctionnalités développées
Ce programme Java lit un fichier CSV contenant des données numériques et calcule pour chaque colonne :
* La moyenne (arrondie à deux décimales)
* La valeur minimale
* La valeur maximale

Le programme est structuré en respectant les principes de la programmation orientée objet, avec une classe principale (`Main`) et une classe dédiée à la logique d'analyse (`AnalyseurCSV`).

## Instructions d'exécution
1. S'assurer d'avoir un fichier nommé `donnees.csv` à la racine du projet, contenant des données numériques séparées par des virgules.
2. Lancer la classe `Main.java` depuis votre environnement de développement (IDE).
3. Les résultats s'afficheront proprement dans la console.
